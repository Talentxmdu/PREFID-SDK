## Get Domain Schema

Get the schema for a specific domain.

<CodeGroup>

```bash Request
curl -X GET "https://api.prefid.dev/v1/domains/music_preferences/schema" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```json Response
{
  "success": true,
  "data": {
    "domain": "music_preferences",
    "version": "1.0",
    "properties": {
      "favorite_artists": {
        "type": "array",
        "items": { "type": "string" },
        "description": "List of favorite artists"
      },
      "genres": {
        "type": "array",
        "items": { "type": "string" },
        "description": "Preferred music genres"
      },
      "streaming_service": {
        "type": "string",
        "enum": ["Spotify", "Apple Music", "YouTube Music", "Other"],
        "description": "Primary music streaming service"
      },
      "listening_contexts": {
        "type": "object",
        "description": "Music preferences by context (work, workout, etc.)"
      }
    }
  }
}
```

</CodeGroup>

## Domain Statistics

Get statistics for a specific domain.

<CodeGroup>

```bash Request
curl -X GET "https://api.prefid.dev/v1/domains/music_preferences/stats" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```json Response
{
  "success": true,
  "data": {
    "domain": "music_preferences",
    "atom_count": 23,
    "last_updated": "2024-12-19T10:30:00Z",
    "confidence_avg": 0.87,
    "sources": {
      "spotify_history": 15,
      "explicit_preference": 5,
      "inferred": 3
    }
  }
}
```

</CodeGroup>
